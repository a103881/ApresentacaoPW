Perguntas Frequentes

Como posso reportar uma situação?

Para reportar uma situação, deve iniciar sessão ou criar uma conta utilizando o seu email da Google. Através dessa conta, poderá aceder aos relatórios existentes e criar os seus próprios.

Posso acompanhar o estado do meu relatório?

Na mesma página onde cria os relatórios, poderá consultar o estado de cada um dos seus relatórios submetidos.

Que tipo de situações posso reportar?

Pode reportar qualquer tipo de situação que considere relevante ou que necessite de intervenção.

Os meus relatórios são anónimos?

Para submeter um relatório é necessário criar uma conta, pelo que os relatórios não são anónimos.

O que acontece depois de eu reportar uma ocorrência?

Após a submissão do relatório, a nossa equipa de peritos irá verificar se a ocorrência é válida. Caso se confirme, será enviada uma equipa para resolver o problema com a maior brevidade possível.
